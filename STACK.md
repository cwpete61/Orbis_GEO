# STACK

## Core Technologies
- **Python 3.8+**: Used for backend logic, data processing, and scoring engines.
- **Bash/Shell Scripts**: Used for installation (`install.sh`) and uninstallation (`uninstall.sh`).

## Runtime Environment
- **Claude Code CLI**: The host environment for the skills and agents.
- **Git**: Required for installation and version management.

## Python Libraries
- **BeautifulSoup4 / lxml**: For HTML parsing and content extraction.
- **Requests**: For HTTP page fetching.
- **ReportLab**: For generating professional PDF reports with charts and visualizations.
- **Playwright (Optional)**: For automated browser interactions and high-quality screenshots.

## Data Formats
- **JSON-LD**: For structured data and schema markup.
- **Markdown**: For agent instructions, skill definitions, and report output.
- **SVG**: Used for assets like the project banner.
